🏎️ SectorTalkF1

Guide complet — Gestion, publication et rendre le blog 100% fonctionnel

📋 ÉTAPE 1 — Utiliser le blog en local

1 Dézipper le fichier

Dézippe SectorTalkF1_FINAL.zip dans un dossier (ex: Bureau/SectorTalkF1)

2 Ouvrir le blog

Double-clic sur sectortalkf1_home.html → Chrome ou Firefox

⚠️ Utilise Chrome ou Firefox. Safari peut bloquer le localStorage.
3 Ouvrir le back-office

Ouvre sectortalkf1_backoffice.html dans un autre onglet.

  • Tout ce que tu modifies dans le BO apparaît immédiatement sur le blog
  • Ctrl+S = Sauvegarder · Ctrl+N = Nouvel article · Ctrl+E = Export JSON
4 Sauvegarder tes données
⚠️ IMPORTANT : Exporte tes données régulièrement ! Si tu vides les cookies → tout perdu.

Back-office → "Export & Liaison Blog" → "Télécharger tout (JSON)" → garde ce fichier précieusement.

🌐 ÉTAPE 2 — Mettre le blog en ligne (Netlify)

GRATUIT Ton blog visible sur Internet en 5 minutes.

1 Créer un compte Netlify

Va sur netlify.com → "Sign up" → avec GitHub ou email.

2 Déployer en drag & drop

Sur ton dashboard Netlify → glisse le dossier SectorTalkF1/ dans la zone "Drag and drop".

Netlify génère une URL type : https://random-name.netlify.app

3 Domaine personnalisé (optionnel)

Netlify → Site settings → Domain management → "Add custom domain"

  • Domaine gratuit Netlify : ton-nom.netlify.app
  • Domaine personnalisé : sectortalkf1.fr (~10€/an sur OVH ou Gandi)
💡 Après chaque modification : retélécharge le dossier modifié sur Netlify (glisse-dépose à nouveau). Le blog est mis à jour instantanément.

✉️ ÉTAPE 3 — Newsletter réelle (Brevo)

GRATUIT 300 emails/jour, contacts illimités.

1 Créer un compte Brevo

Va sur brevo.com → Crée un compte gratuit.

2 Créer une liste de contacts

Dashboard → Contacts → Listes → "Créer une liste" → note l'ID (ex: 3).

3 Créer une clé API

Paramètres (icône en haut à droite) → API Keys → "Générer une nouvelle clé" → Copie-la.

4 Configurer le fichier newsletter

Ouvre sectortalkf1_newsletter.html avec Notepad++ ou VSCode.

Cherche VOTRE_CLE_API_BREVO → remplace par ta clé.

Cherche listId: 0 → remplace par ton ID de liste (ex: listId: 3).

Sauvegarde et redéploie sur Netlify.

▶️ ÉTAPE 4 — Vidéos YouTube

GRATUIT Intégrer tes vidéos YouTube dans la page Médias.

1 Uploader tes vidéos sur YouTube

Crée une chaîne YouTube "SectorTalkF1" → Upload tes vidéos.

2 Intégrer dans le blog

Page Médias → onglet Vidéos → Clique sur une carte → colle l'ID YouTube.

L'ID est la partie après ?v= dans l'URL YouTube.

Ex: youtube.com/watch?v=dQw4w9WgXcQ → ID = dQw4w9WgXcQ

🎙️ ÉTAPE 5 — Podcast réel

GRATUIT Héberger et jouer tes vrais podcasts.

1 Héberger les MP3

Options gratuites : Podbean (gratuit 5h), Anchor/Spotify for Podcasters (illimité), ou GitHub Pages (si fichiers <100MB).

2 Ajouter l'URL dans le back-office

Back-office → Podcast → Ajoute l'épisode avec l'URL du fichier MP3.

Le lecteur de la page Médias lira le vrai audio automatiquement.

📊 ÉTAPE 6 — Analytics (statistiques)

GRATUIT Savoir combien de visiteurs tu as.

1 Google Analytics

Va sur analytics.google.com → Créer une propriété → copie l'ID (ex: G-XXXXXXXXXX).

Dans chaque fichier .html, cherche G-XXXXXXXXXX → remplace par ton vrai ID → redéploie.

2 Alternative RGPD : Plausible

Plausible.io = analytics sans cookies (RGPD friendly). Gratuit en auto-hébergement.

Dans les pages HTML, décommente le script Plausible et remplace sectortalkf1.fr par ton domaine.

🔧 MODIFIER LE BLOG — Référence rapide

Via le back-office (sans code)

  • Nouvel article → Ctrl+N ou bouton "✍️ Nouvel article"
  • Modifier les points F1 → "Classements & Résultats" → cases éditables
  • Saisir un résultat de course → points recalculés automatiquement
  • Modifier un pilote → "Gestion Pilotes" → ✏️ pour la bio
  • Exporter toutes les données → Ctrl+E

Modifier la navigation (ajouter une page)

Ouvre stf1_nav.js avec Notepad++ → trouve NAV_LINKS → ajoute une ligne → sauvegarde → s'applique à toutes les pages.

Modifier une page (contenu)

Ouvre le .html avec VSCode → Ctrl+F pour chercher → modifie → sauvegarde → rechargement automatique.
